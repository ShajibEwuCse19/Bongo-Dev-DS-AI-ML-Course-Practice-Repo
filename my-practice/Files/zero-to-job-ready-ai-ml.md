# Zero To Job Ready AI/ML Roadmap

এই guide একদম শুরু থেকে job-ready হওয়া পর্যন্ত roadmap হিসেবে লেখা হয়েছে।  
Goal হলো:

- zero level থেকে শুরু করা
- project-ready হওয়া
- internship / junior / entry-level AI/ML/Data Science position-এর জন্য প্রস্তুত হওয়া

---

## 1. Final Target কী

তোমার target হওয়া উচিত এমন level-এ যাওয়া যেখানে তুমি:

- Python-এ code লিখতে পারো
- data clean করতে পারো
- EDA করতে পারো
- ML model train/evaluate করতে পারো
- 3-4টা ভালো project দেখাতে পারো
- SQL query লিখতে পারো
- basic deployment করতে পারো
- interview-এ নিজের কাজ explain করতে পারো

এটাই entry-level job-ready stage।

---

## 2. Full Journey Overview

পুরো roadmap:

`Basic Computer Skill -> Python -> Math/Stats -> Data Analysis -> SQL -> Machine Learning -> Projects -> Deployment -> Portfolio -> Interview -> Job Apply`

---

## 3. Stage 1: Zero Level Foundation

সময়: `2-3 weeks`

যদি তুমি একদম শুরুতে থাকো, আগে programming comfort তৈরি করো।

### কী শিখবে

- computer file/folder management
- terminal basic
- VS Code / Jupyter Notebook use
- GitHub account create
- Python install/use

### Python Basic

- variable
- data type
- if else
- loop
- function
- list
- dictionary
- tuple
- set
- string
- input/output

### Goal

এই stage শেষে তুমি ছোট Python script লিখতে পারবে।

### Mini Practice

- calculator
- number guessing
- list operations
- file read/write

## 4. Stage 2: Python for Data Work

সময়: `3-4 weeks`

এখন general Python থেকে data-related Python-এ যাও।

### কী শিখবে

- NumPy
- Pandas
- Matplotlib
- Seaborn

### কী কী করতে পারতে হবে

- CSV load
- column select
- filter
- sort
- groupby
- missing value check
- basic charts

### Practice Task

- sales data analysis
- student performance analysis
- housing dataset basic analysis

### Goal

একটি dataset load করে clean summary দিতে পারতে হবে।

## 5. Stage 3: Math and Statistics for ML

সময়: `2-4 weeks`

ML-এর জন্য খুব advanced math শুরুতে দরকার না, but কিছু concept strong থাকা দরকার।

### Must Know Topics

- mean
- median
- mode
- variance
- standard deviation
- probability basic
- normal distribution basic
- correlation
- covariance basic idea
- hypothesis testing basic
- bias-variance tradeoff

### কেন দরকার

- data distribution বুঝতে
- scaling বুঝতে
- evaluation বুঝতে
- overfitting/underfitting বুঝতে

### Goal

metric আর data behavior intuition তৈরি হওয়া।

## 10. Stage 8: Project Ready হওয়া

সময়: `6-10 weeks`

### Minimum Project Count

- `2টা classification project`
- `1টা end-to-end deployment project`

### Project Structure

প্রতিটি project এ থাকবে:

1. problem statement
2. business objective
3. dataset source
4. preprocessing
5. EDA
6. baseline model
7. improved model
8. evaluation
9. error analysis
10. improvement plan

### Best Beginner Project Ideas

- house price prediction
- telecom churn prediction
- employee attrition prediction
- credit risk classification
- customer segmentation
- sales forecasting
- sentiment analysis

### Bangladesh Market Friendly Project Ideas

- telecom churn
- recharge behavior prediction
- customer complaint classification
- delivery demand prediction
- loan approval prediction
- e-commerce recommendation basic

## 11. Stage 9: Job Ready Projects কেমন হবে

Project শুধু notebook হলে চলবে না। Job-ready project-এ কিছু extra জিনিস থাকতে হবে।

### Job-Ready Project Checklist

- clean README
- proper problem statement
- business use case
- clear dataset explanation
- reproducible notebook/script
- requirements.txt
- model save file
- sample input/output
- result visualization
- future improvement section

### Best Format

প্রতিটি project repo:

`project-name/`

- `README.md`
- `notebooks/`
- `src/`
- `data/` অথবা data source link
- `models/`
- `requirements.txt`

## 22. Short Final Advice

তোমার জন্য best path হবে:

`Learn -> Practice -> Build -> Explain -> Deploy -> Apply`
